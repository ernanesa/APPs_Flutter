# Política de Privacidade - White Noise

**Última atualização:** 17/01/2026

## 1. Informações que Coletamos

### 1.1. Dados fornecidos pelo usuário
- Preferências de sons e configurações do app (armazenadas localmente)

### 1.2. Dados coletados automaticamente
- Identificadores de publicidade e dados de diagnóstico via AdMob

## 2. Como usamos seus dados
- Para oferecer a funcionalidade do app e exibir anúncios

## 3. Compartilhamento de dados
- Compartilhamos dados com Google AdMob para exibição de anúncios

## 4. Armazenamento e segurança
- Dados de configuração são armazenados localmente no dispositivo

## 5. Seus direitos
- Você pode desinstalar o app para remover dados locais
